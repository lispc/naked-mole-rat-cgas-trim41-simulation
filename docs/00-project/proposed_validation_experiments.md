# 验证"变构特异性"假设的计算实验方案

## 核心假设

```
4mut 位点（C-端中段）
    ↓ 长程变构效应
cGAS N-端构象改变
    ↓ 改变识别特异性
TRIM41 界面（N-端/中段）
```

**预测**: 
- 亲和力（ΔG_bind）基本不变 ← 突变扫描已支持
- 特异性（结合模式/动力学）改变 ← 待验证
- 变构路径存在（4mut ↔ N-端动态耦合）← 待验证

---

## 优先级排序

### 🔴 P0：等 Hgal MD 完成后立即做

#### 1. 动态交叉相关矩阵（DCCM）
**目的**: 直接检测 4mut 位点与界面残基的**动态耦合**
- 方法: `pytraj`/`MDAnalysis` 计算 Cα 原子对的运动相关性
- 预期: WT 中 463/511/527/530 与界面残基（213/242/247）有正相关；4mut 中这种耦合改变或消失
- 工作量: 1-2 小时

#### 2. 主成分分析（PCA）
**目的**: 看 WT vs 4mut 是否占据不同的构象子空间
- 方法: 对 Cα 坐标做 PCA，投影到 PC1/PC2
- 预期: 两个体系在 PC 空间中分离，N 端区域贡献最大方差
- 工作量: 2-3 小时

#### 3. RMSF + 差异 RMSF
**目的**: 精确定位哪些区域柔性改变
- 方法: 计算每个残基的 RMSF，做 ΔRMSF = RMSF_4mut − RMSF_WT
- 预期: N 端（200-220）ΔRMSF 显著；4mut 位点本身变化小
- 工作量: 1 小时

#### 4. 氢键网络追踪
**目的**: 看 4mut 是否打断/建立了长程氢键，从而传递变构信号
- 方法: `MDAnalysis` 统计残基间氢键占有率
- 特别关注: 4mut 位点 ↔ 中间桥接残基 ↔ 界面残基
- 预期: 存在一条或多条氢键路径在 WT 和 4mut 间差异显著
- 工作量: 3-4 小时

---

### 🟡 P1：可在 M3 Pro 上现在做

#### 5. Rosetta FastRelax + 界面能量
**目的**: 比当前的 Pack+Min 更准确地评估变构效应对界面的影响
- 方法: 对 WT 和 4mut 复合物分别做 FastRelax（允许骨架放松）
- 然后计算 I_sc，比较 ΔΔG
- 当前局限: Pack+Min 固定骨架，FastRelax 可能让 N 端弛豫到更低能量状态
- 预期: FastRelax 后 I_sc 可能仍有微小变化，但主要信号在 N 端 RMSD
- 工作量: 2-3 小时（M3 Pro，2 个体系）

#### 6. 接触图差异分析（AF3 结构）
**目的**: 用静态结构推断动态接触变化
- 方法: 对 WT 和 4mut 的 5 个 AF3 model 分别计算残基接触图（<8Å）
- 比较 WT vs 4mut 的接触丢失/获得
- 预期: N 端区域接触模式改变；4mut 位点周围局部接触也改变
- 工作量: 2 小时

#### 7. MM-GBSA 结合自由能分解
**目的**: 定量验证"亲和力不变"
- 方法: 用 `pygb` 或 `MMPBSA.py` 对 WT 和 4mut 对接结构做 MM-GBSA
- 分解到每个残基的贡献（per-residue decomposition）
- 预期: 总 ΔG_bind 差异 < 2 kcal/mol；但界面残基的贡献分布不同
- 工作量: 3-4 小时

---

### 🟢 P2：需要额外数据/资源

#### 8. Hsap MD（WT + 4mut）
**目的**: 验证机制在 Human 体系是否保守
- 已在计划内，等 3090 空闲后跑
- 4 个系统 × 3 rep × 200ns = 大量 GPU 时间

#### 9. 集合变异分析（ENSEMBLE）
**目的**: 用 AF3 的 5 个 model 做初步 ensemble 分析
- 方法: 计算 WT 和 4mut  ensemble 的 N 端构象分布差异
- 用 KL 散度或 RMSD 分布量化
- 工作量: 1-2 小时

#### 10. 变构路径预测（AlloSigMA 或类似）
**目的**: 计算方法预测 4mut → 界面的信号传递路径
- 方法: 用 `AlloSigMA` 或 `pyANM`（各向异性网络模型）
- 输入: 单个 PDB 结构
- 输出: 残基间机械耦合强度，预测变构热点
- 工作量: 2-3 小时（需安装额外包）

## 最强的验证信号

如果能同时观察到以下 4 个信号，假设基本成立：

1. ✅ **I_sc 不变**（Rosetta 突变扫描）— 已有
2. ✅ **4mut 远离界面**（Rosetta 对接）— 已有
3. 🔄 **DCCM 显示动态耦合改变**（MD）— 等 Hgal MD
4. 🔄 **N 端构象子空间分离**（PCA）— 等 Hgal MD

第 3、4 个是**关键缺失证据**，也是 MD 分析的核心价值。
