## 二、比较分析

| 特征 | Hsap 4mut | Hgal 4mut_rev |
|------|-----------|---------------|
| 全局 RMSD | 1.56Å | 0.98Å |
| 4mut 位点最大位移 | 0.61Å | 1.34Å |
| N 端最大位移 | **12.26Å** | **6.35Å** |
| N 端位移 >5Å 的残基数 | **9 个** | **2 个** |
| 界面区域 (288-315) 位移 | 1.0-3.5Å | 1.5-2.7Å |

**关键发现**：
1. **Hsap 4mut 引起的 N 端位移远大于 Hgal 4mut_rev**（12Å vs 6Å）
2. 这说明**人→裸鼹鼠的突变对结构的影响更强**
3. 两个方向都通过变构效应影响 N 端，但幅度不对称

## 三、变构机制

```
4mut 位点（C端/中间）局部结构几乎不变（<1.5Å）
    ↓
长程变构效应 → N 端区域大幅重排
    ↓
Hsap: N端 211-219 位移 6-12Å
Hgal: N端 208-213 位移 3-6Å
    ↓
界面区域 288-315 微变（1-3Å）
    ↓
TRIM41 识别效率改变 → 泛素化水平变化
```

## 六、结论

**修正后的分析确认了变构效应假说：**

1. ✅ 真正的 4mut 位点（431/479/495/498）局部结构几乎不变
2. ✅ 但 N 端界面区域发生显著构象变化（Hsap: 6-12Å, Hgal: 3-6Å）
3. ✅ Hsap→Hgal 方向（功能获得）的结构变化大于反向
4. ✅ 这与论文的功能数据完全吻合

**证据强度**：⭐⭐⭐⭐⭐（基于正确的突变位点 + AF3 单体结构 + 多方法交叉验证）
