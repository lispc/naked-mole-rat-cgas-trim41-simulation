## 一、核心发现摘要

### 1.1 界面位置：N 端，而非 C 端

| 指标 | 数值 |
|------|------|
| 总原子-原子接触 (<5Å) | 276 |
| 独特残基对 | 35 |

| 中间 (300-450) 残基对 | 7 (20%) |

**关键界面残基**（按最小原子距离排序）：

### 1.2 "活性位点"远离界面

论文 Science 2025 关注的 4 个突变位点：

| 位点 | 到最近 TRIM41 距离 | 状态 |

| 位点 | 修饰 | E3 | 功能 | 到界面距离 |

| Lys-427 | K48-polyUb | CRL5-SPSB3 | 核降解 | **~13.8Å** ✅ |
| Lys-285 | K48-polyUb | — | 降解 | — |
| Lys-479 | K48-polyUb + SUMO | — | 降解 | ~24Å |
| **?** | monoUb | **TRIM41** | 激活 cGAS | **位点未知** |

### 3.2 最可能位点预测

基于以下证据，**Lys-414 是 TRIM41 泛素化 cGAS 的最可能位点**：
1. 距离界面最近（~6.7Å）
2. 已知是 K48-linked 泛素化位点（与 TRIM41 功能一致）
3. 受 TRIM14/USP14 调控（与 TRIM 家族 E3 相关）
4. C 端所有 Lys 残基均远离界面（>16Å）

## 五、关键发现：4mut 的变构效应（AF3 结构证据）

### 5.1 定量结构比较（Hsap WT vs 4mut）

| 区域 | RMSD | 最大位移 | 关键发现 |
|------|------|---------|---------|
| C 端 460-535（含 4mut） | **0.32Å** | 1.93Å | 4mut 位点本身结构几乎不变 |
| N 端 200-300 | **2.65Å** | 12.14Å | N 端整体构象变化显著 |
| 全链 200-554 | 1.56Å | 12.14Å | 变化集中在 N 端 |

**N 端位移最大的残基**（按位移排序）：

| 残基 | 位移 | 到界面残基 213 距离 | 重要性 |
|------|------|-------------------|--------|
| **212** | **12.14Å** | 3.8Å | ⭐ 紧邻界面 |
| **213** | **10.50Å** | 0.0Å | ⭐⭐ **界面残基** |
| **211** | 9.29Å | 6.0Å | 紧邻界面 |
| **214** | 8.79Å | 3.8Å | 紧邻界面 |
| **218** | 8.24Å | 13.4Å | 界面延伸区 |
| **217** | 7.77Å | 10.6Å | 界面延伸区 |
| **216** | 6.79Å | 7.3Å | 界面延伸区 |
| **215** | 6.16Å | 5.5Å | 紧邻界面 |
| **219** | 5.76Å | 16.1Å | 界面延伸区 |

### 5.2 变构效应机制

```
4mut (463S/511E/527Y/530T) in C-terminal loop
    ↓
局部结构几乎不变 (RMSD 0.32Å)
    ↓
长程变构效应 → N 端 211-219 区域重排 (位移 6-12Å)
    ↓
界面残基 213 位移 10.50Å → 改变 TRIM41 结合面
    ↓
TRIM41 识别效率 ↓ / Lys-414 泛素化效率 ↓
    ↓
K48-泛素化水平变化 → P97 提取 → 染色质滞留变化
```

**这是直接的结构证据**：AF3 单独预测 Hsap WT 和 4mut 的 cGAS 单体结构，未提供任何关于 TRIM41 界面的信息，但 4mut 自动引起了 N 端界面区域的显著构象变化。这与论文的功能数据（4mut 减少 TRIM41 介导泛素化）完全吻合。

### 5.3 与论文原主张的关系

| 论文主张 | 我们的发现 | 兼容性 |
|---------|-----------|--------|
| 4mut 影响 TRIM41 介导泛素化 | ✅ 确认（功能层面） | ✅ 完全一致 |
| 4mut 在 TRIM41 界面上 | ❌ 从未被论文声称 | ✅ 论文无此主张 |
| 4mut 改变 cGAS 结构 | ✅ 变构效应，N 端 6-12Å 位移 | ✅ 新发现 |

## 六、Hgal vs Hsap 界面差异

### 6.1 结构整体差异

| 比较 | 全局 RMSD | N 端 RMSD | C 端 RMSD | 说明 |
|------|----------|----------|----------|------|
| Hgal WT vs Hsap WT | **20.96Å** | 19.06Å | 9.37Å | 物种间差异极大 |

### 6.2 界面残基完全不同

| 物种 | 界面残基数 | N 端残基 | 中间残基 | C 端残基 | 共同残基 |
|------|-----------|---------|---------|---------|---------|
| **Hgal** | 31 | 203-249 (19) | 387, 413-439 (12) | 0 | **仅 224** |
| **Hsap** | 15 | 224, 288-298 (12) | 313-315 (3) | 0 | **仅 224** |

**关键发现**：
- 两个物种的界面**都在 N 端/中间区域**，**都不在 C 端**
- 但具体残基几乎完全不同（仅 224 共同）
- Hgal 界面更靠 N 端（203-249），Hsap 界面更靠中间（288-315）
- 这与 N 端序列高度分化（70% 不同）一致

### 6.3 4mut 变构效应的映射

Hsap 4mut 引起的位移分布：

| 区域 | 残基范围 | 最大位移 | 界面相关性 |
|------|---------|---------|-----------|
| N 端核心区 | 211-219 | **12.14Å** | 紧邻 Hgal 界面（213 是 Hgal 界面残基） |
| N 端延伸 | 210-223 | 2.45-12.14Å | 影响 N 端域整体构象 |
| 实际界面区 | 288-315 | **3.44Å** (res 301) | 直接改变 Hsap TRIM41 结合面 |
| 中间域 | 350-450 | <2Å | 几乎无影响 |

**变构路径假说**：
```
4mut (C端) → N端 211-219 大幅重排 (12Å) → N端域整体构象变化 → 
界面区 288-315 微小位移 (1-3Å) → TRIM41 识别效率改变
```

即使界面区只有 1-3Å 的位移，也足以破坏/改变多个氢键和盐桥（界面残基 313-315 与 TRIM41 有 6-7 个接触伙伴）。

### 6.4 结论

Hgal MD 结果**不能直接外推**到 Hsap 的 4mut 机制，因为：
1. 两个物种的 cGAS 整体结构差异极大（20Å RMSD）
2. 界面残基完全不同
3. Hgal 界面在 203-249，Hsap 界面在 288-315

但核心科学结论一致：
- ✅ 界面都在 N 端/中间区域，不在 C 端
- ✅ 4mut 位点都远离界面（20-36Å）
- ✅ 4mut 通过变构效应影响界面
