### 2. 核心科学发现：4mut 的变构效应（AF3 结构证据）

Hsap WT vs 4mut AF3 单体结构比较：

| 区域 | RMSD | 最大位移 | 意义 |
|------|------|---------|------|
| C 端 460-535（含 4mut） | **0.32Å** | 1.93Å | 4mut 本身结构不变 |
| N 端 211-219 | **2.65Å** | **12.14Å** | N 端构象大幅改变 |
| Hsap 界面区 288-315 | ~1-3Å | 3.44Å | 界面微变，足以改变相互作用 |

**变构机制**：
```
4mut (C端) → N端 211-219 大幅重排 (12Å) → N端域构象变化 →
界面区 288-315 微变 (1-3Å) → TRIM41 识别效率↓ → 泛素化↓
```

这是**直接的结构证据**：AF3 单独预测 WT 和 4mut，未提供任何 TRIM41 信息，但 4mut 自动引起 N 端界面区域构象变化。
